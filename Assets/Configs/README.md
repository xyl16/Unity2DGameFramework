# 配置表文件夹说明

此文件夹用于存放CSV格式的配置表文件。

## 文件夹结构

```
Assets/Configs/
├── Item/          # 道具配置表
├── Character/     # 角色配置表
├── Level/         # 关卡配置表
└── ...            # 其他配置表
```

## CSV文件格式规范

配置表应遵循以下格式：

```
字段说明,道具名称,道具ID,道具类型,稀有度,描述
字段类型,string,int,int,string,string

示例道具,1001,1,1,普通
高级武器,1002,1,3,稀有
```

- 第一行以"字段说明"开头，后面跟字段描述
- 第二行以"字段类型"开头，后面跟字段名称和类型（每两个字段为一组）
- 字段类型支持: int, float, bool, string
- 从分隔线后开始为数据行

## 使用方法

1. 在Unity中通过 `Tools/配置表导出工具` 打开导出工具
2. 选择源文件夹（默认: Assets/Configs）
3. 选择输出文件夹（默认: Assets/Resources/Configs）
4. 选择导出格式（二进制/JSON/Protobuf）
5. 点击"导出所有配置表"按钮
