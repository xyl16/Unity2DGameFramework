# 网络和配置系统使用说明

## 一、配置表系统

### 1.1 CSV配置文件格式

CSV配置文件需要遵循以下格式：

```
字段说明,字段名,字段类型
字段说明文本,fieldName,fieldType
...

数据行...
```

前3行固定格式：
- 第1行：字段说明（中文描述）
- 第2行：字段名（变量名）
- 第3行：字段类型（int/string/float/bool）

从第4行开始是数据行。

### 1.3 创建配置类

```csharp
using System;
using System.Collections.Generic;

public class ItemConfig : BaseConfigRow
{
    public int id;
    public string name;
    public int type;      // 1:武器 2:消耗品 3:材料
    public int rarity;    // 1:普通 2:稀有 3:史诗 4:传说
    public string baseAttr;
    public string description;
    public int price;
    public int stackLimit;

    public override void Parse(Dictionary<string, string> data)
    {
        id = GetInt(data, "id");
        name = GetString(data, "name");
        type = GetInt(data, "type");
        rarity = GetInt(data, "rarity");
        baseAttr = GetString(data, "baseAttr");
        description = GetString(data, "description");
        price = GetInt(data, "price");
        stackLimit = GetInt(data, "stackLimit");
    }
}
```

### 1.4 加载和使用配置

#### 方式一：直接获取配置值

```csharp
// 加载CSV文件
ConfigTableManager.Instance.LoadConfigFromCSV("ItemConfig", "Assets/Config/ItemConfig.csv");

// 直接获取配置值
int price = ConfigTableManager.Instance.GetIntValue("ItemConfig", 1, "price", 0);
string name = ConfigTableManager.Instance.GetStringValue("ItemConfig", 1, "name", "");
float value = ConfigTableManager.Instance.GetFloatValue("ItemConfig", 1, "value", 0f);
bool enabled = ConfigTableManager.Instance.GetBoolValue("ItemConfig", 1, "enabled", false);
```

#### 方式二：使用强类型配置

```csharp
// 加载为强类型配置
ConfigTableManager.Instance.LoadConfig<ItemConfig>("ItemConfig");

// 获取单个配置
ItemConfig item = ConfigTableManager.Instance.GetConfig<ItemConfig>(1);
if (item != null)
{
    Debug.Log($"物品: {item.name}, 价格: {item.price}");
}

// 获取所有配置
var allItems = ConfigTableManager.Instance.GetAllConfig<ItemConfig>();
foreach (var kvp in allItems)
{
    Debug.Log($"ID: {kvp.Key}, 名称: {kvp.Value.name}");
}
```

#### 方式三：从AssetBundle加载

```csharp
// 配置文件打包到AssetBundle时，使用TextAsset类型
ConfigTableManager.Instance.LoadConfigFromAB("ItemConfig", "ItemConfig");
```

## 二、网络通信系统

### 2.1 基础网络使用

#### NetworkManager（基础版）

```csharp
// 注册消息处理器
NetworkManager.Instance.RegisterMessageHandler(MessageID.LOGIN_RESPONSE, HandleLoginResponse);

// 连接服务器
ConfigManager.AppConfig config = ConfigManager.Instance.GetConfig();
NetworkManager.Instance.Connect(config.serverIP, config.serverPort);

// 发送消息
var loginRequest = new LoginRequest
{
    username = "player1",
    password = "password123",
    deviceId = SystemInfo.deviceUniqueIdentifier
};
NetworkManager.Instance.SendMessage(MessageID.LOGIN_REQUEST, loginRequest);

// 处理响应
private void HandleLoginResponse(byte[] data)
{
    string json = System.Text.Encoding.UTF8.GetString(data);
    LoginResponse response = JsonUtility.FromJson<LoginResponse>(json);
    Debug.Log($"登录结果: {response.code}, 消息: {response.message}");
}
```

### 2.2 安全网络使用

#### SecureNetworkManager（推荐使用）

```csharp
// 设置序列化方式
SecureNetworkManager.Instance.SetSerializeType(MessageSerializer.SerializeType.JSON);

// 注册消息处理器（泛型版本，自动反序列化）
SecureNetworkManager.Instance.RegisterMessageHandler<LoginResponse>(
    MessageID.LOGIN_RESPONSE,
    (response) => {
        Debug.Log($"登录结果: {response.code}, Token: {response.token}");
        if (response.code == 0)
        {
            // 登录成功
            OnLoginSuccess(response);
        }
    }
);

// 连接服务器（启用加密）
SecureNetworkManager.Instance.Connect("127.0.0.1", 8888, enableEncryption: true);

// 监听连接事件
SecureNetworkManager.Instance.OnConnected += () => {
    Debug.Log("已连接到服务器");
    // 发送登录请求
    SendLoginRequest();
};

SecureNetworkManager.Instance.OnDisconnected += () => {
    Debug.Log("已断开连接");
};

// 发送登录请求
private void SendLoginRequest()
{
    var request = new LoginRequest
    {
        username = "player1",
        password = "password123",
        deviceId = SystemInfo.deviceUniqueIdentifier,
        platform = Application.platform.ToString(),
        version = 1
    };
    SecureNetworkManager.Instance.SendMessage(MessageID.LOGIN_REQUEST, request);
}
```

### 2.3 定义网络消息

所有网络消息需要继承自 `ProtoMessage` 基类：

```csharp
[Serializable]
public class LoginRequest : ProtoMessage
{
    public override ushort MessageId { get { return 1001; } }

    public string username;
    public string password;
    public string deviceId;
    public string platform;
    public int version;
}

[Serializable]
public class LoginResponse : ProtoMessage
{
    public override ushort MessageId { get { return 1002; } }

    public int code;           // 0:成功 其他:错误码
    public string message;
    public string token;
    public long userId;
    public PlayerInfo playerInfo;
}

[Serializable]
public class PlayerInfo
{
    public long userId;
    public string nickname;
    public int level;
    public long exp;
    public int vipLevel;
    public long coin;
    public long diamond;
}
```

## 三、序列化系统

### 3.1 支持的序列化方式

框架支持以下序列化方式：

| 序列化类型 | 特点 | 适用场景 |
|----------|------|---------|
| JSON | 可读性好，通用性强 | 调试、简单数据 |
| ProtoBuf | 二进制，体积小，速度快 | 大数据量传输 |
| Binary | 二进制，自定义实现 | 特殊需求 |
| XML | 可读性好，支持复杂结构 | 配置文件、复杂结构 |

### 3.2 切换序列化方式

```csharp
// 设置为JSON序列化
SecureNetworkManager.Instance.SetSerializeType(MessageSerializer.SerializeType.JSON);

// 设置为ProtoBuf序列化
SecureNetworkManager.Instance.SetSerializeType(MessageSerializer.SerializeType.ProtoBuf);

// 设置为二进制序列化
SecureNetworkManager.Instance.SetSerializeType(MessageSerializer.SerializeType.Binary);
```

### 3.3 数据压缩

```csharp
// 启用压缩
SecureNetworkManager.Instance.EnableCompression = true;

// 数据会自动压缩传输
```

### 4.2 手动加密示例

```csharp
CryptoManager crypto = CryptoManager.Instance;

// 生成密钥
byte[] key = crypto.GenerateRandomKey(32);
byte[] iv = crypto.GenerateRandomIV(16);

// AES加密
string plainText = "Hello, Server!";
byte[] plainBytes = System.Text.Encoding.UTF8.GetBytes(plainText);
byte[] encrypted = crypto.AESEncrypt(plainBytes, key, iv);

// AES解密
byte[] decrypted = crypto.AESDecrypt(encrypted, key, iv);
string decryptedText = System.Text.Encoding.UTF8.GetString(decrypted);

// 哈希计算
string md5 = crypto.MD5Hash("data");
string sha256 = crypto.SHA256Hash("data");

// HMAC签名
string hmac = crypto.HMACSHA256("message", "secret");
```

## 五、完整示例

### 5.1 登录流程完整示例

```csharp
using UnityEngine;

public class LoginController : MonoBehaviour
{
    private void Start()
    {
        // 初始化网络
        InitializeNetwork();

        // 自动登录（如果保存了token）
        string savedToken = PlayerPrefs.GetString("auth_token", "");
        if (!string.IsNullOrEmpty(savedToken))
        {
            AutoLogin(savedToken);
        }
    }

    private void InitializeNetwork()
    {
        // 设置序列化方式
        SecureNetworkManager.Instance.SetSerializeType(MessageSerializer.SerializeType.JSON);

        // 启用加密
        SecureNetworkManager.Instance.EnableEncryption = true;

        // 注册消息处理器
        RegisterMessageHandlers();

        // 监听连接事件
        SecureNetworkManager.Instance.OnConnected += OnConnected;
        SecureNetworkManager.Instance.OnDisconnected += OnDisconnected;
        SecureNetworkManager.Instance.OnError += OnError;
    }

    private void RegisterMessageHandlers()
    {
        // 登录响应
        SecureNetworkManager.Instance.RegisterMessageHandler<LoginResponse>(
            MessageID.LOGIN_RESPONSE,
            HandleLoginResponse
        );

        // 角色列表响应
        SecureNetworkManager.Instance.RegisterMessageHandler<GetCharacterListResponse>(
            MessageID.GET_CHARACTER_LIST_RESPONSE,
            HandleCharacterList
        );
    }

    private void OnConnected()
    {
        Debug.Log("已连接到服务器");
        UIManager.Instance.ShowConnecting(false);
    }

    private void OnDisconnected()
    {
        Debug.Log("已断开连接");
        UIManager.Instance.ShowConnecting(false);
        ShowReconnectDialog();
    }

    private void OnError(string error)
    {
        Debug.LogError($"网络错误: {error}");
        UIManager.Instance.ShowToast(error);
    }

    // 账号密码登录
    public void Login(string username, string password)
    {
        var request = new LoginRequest
        {
            username = username,
            password = cryptoManager.MD5Hash(password),  // 密码MD5加密
            deviceId = SystemInfo.deviceUniqueIdentifier,
            platform = Application.platform.ToString(),
            version = Application.version.GetHashCode()
        };

        SecureNetworkManager.Instance.SendMessage(MessageID.LOGIN_REQUEST, request);
    }

    // Token自动登录
    private void AutoLogin(string token)
    {
        // 使用Token登录的实现
    }

    private void HandleLoginResponse(LoginResponse response)
    {
        if (response.code == 0)
        {
            // 登录成功
            Debug.Log($"登录成功: {response.playerInfo.nickname}");

            // 保存Token
            PlayerPrefs.SetString("auth_token", response.token);
            PlayerPrefs.Save();

            // 加载玩家数据
            LoadPlayerData(response.playerInfo);

            // 获取角色列表
            GetCharacterList(response.token);
        }
        else
        {
            // 登录失败
            UIManager.Instance.ShowToast($"登录失败: {response.message}");
        }
    }

    private void LoadPlayerData(PlayerInfo playerInfo)
    {
        Debug.Log($"玩家信息: 等级{playerInfo.level}, 金币{playerInfo.coin}");
        // 更新UI显示
        UIManager.Instance.UpdatePlayerInfo(playerInfo);
    }

    private void GetCharacterList(string token)
    {
        var request = new GetCharacterListRequest
        {
            token = token
        };
        SecureNetworkManager.Instance.SendMessage(MessageID.GET_CHARACTER_LIST_REQUEST, request);
    }

    private void HandleCharacterList(GetCharacterListResponse response)
    {
        if (response.code == 0)
        {
            Debug.Log($"获取到{response.characters.Count}个角色");
            // 显示角色列表
            UIManager.Instance.ShowCharacterList(response.characters);
        }
    }

    private void OnDisable()
    {
        // 清理事件监听
        SecureNetworkManager.Instance.OnConnected -= OnConnected;
        SecureNetworkManager.Instance.OnDisconnected -= OnDisconnected;
        SecureNetworkManager.Instance.OnError -= OnError;
    }
}
```

## 七、最佳实践

### 7.1 配置表管理

1. **使用强类型配置**：避免直接使用字符串字段名，减少错误
2. **缓存配置**：游戏启动时一次性加载所有配置
3. **版本管理**：配置文件名包含版本号，便于管理
4. **热更新**：将配置打包到AssetBundle，支持热更新

### 7.2 网络通信

1. **使用SecureNetworkManager**：比普通NetworkManager更安全
2. **启用加密**：生产环境务必启用加密
3. **心跳检测**：框架自动处理，保持连接活跃
4. **错误处理**：监听OnError事件，优雅处理网络错误
5. **重连机制**：断线后自动重连（框架已实现）

### 7.3 消息定义

1. **继承ProtoMessage**：所有网络消息继承该基类
2. **使用MessageID常量**：避免硬编码消息ID
3. **字段序列化**：使用[Serializable]特性标记
4. **版本兼容**：新版本兼容旧消息格式

### 7.4 序列化选择

1. **开发阶段**：使用JSON，便于调试
2. **生产环境**：使用ProtoBuf，提升性能
3. **大数据传输**：启用压缩
4. **敏感数据**：启用加密

## 八、常见问题

### Q1: CSV文件中文乱码？

A: 确保CSV文件使用UTF-8编码保存。

### Q2: 配置加载失败？

A: 检查CSV格式是否正确，前3行必须符合规范。

### Q3: 网络连接超时？

A: 检查服务器地址和端口，确保服务器可访问。

### Q4: 消息接收不到？

A: 确保消息ID一致，且已注册对应的消息处理器。

### Q5: 加密导致性能问题？

A: 可以根据需要选择性启用加密，非敏感数据可以不加密。
